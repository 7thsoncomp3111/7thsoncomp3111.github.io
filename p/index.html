Voting and Threaded Style

Webapp version
Native Android version
In the following, I'll introduce two commonly found but essential functions of Just Ask, namely Voting and Threaded Style. As a user, one would want to make +ve/-ve vote on questions based on personal preference so that one would see questions popular among the whole user-base. To verify this, the upvotes/downvotes should reflect changes in realtime as users vote on the same question. As a user, one would view the essentials of the questions in a clutter-free interface so that one could select the question of interest among the question bank. To verify this, the interface should consist of the title and relevant statistics such as views. Additional visual elements could be added to reflect the status of the question, such as it is highlighted/closed.
